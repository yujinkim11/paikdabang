## ❔ 구현 목적

React를 이용하여 퍼블리싱, 구조 잡는 것을 연습하기 위해 구현<br/>
swiper, aos 기능을 더 능숙하게 사용하고 싶어 구현

## 💡 주요 기능

Swiper로 배너 제작 (Autoplay 설정)
aos 사용
